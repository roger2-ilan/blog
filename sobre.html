Quem sou eu

Olá, eu sou o Roger Ilan e tenho interesse em tecnologia, computação, programação e principalmente sistemas operacionais.

Sou deficiente visual. Não enxergo desde que nasci, pois tive uma úlcera em um olho e, no outro, descolamento de córnea.

Mesmo assim, com minhas limitações, estou aqui escrevendo esta página.

Consigo usar o computador de forma razoável. Para isso, uso um leitor de tela. Existem vários no mercado, mas os que mais se destacam são o NVDA, para Windows, e o Orca, para Linux e outros sistemas operacionais Unix.

Por que esse site foi criado?

Eu sempre quis criar um site para compartilhar conhecimentos que fui adquirindo no mundo da computação.

Porém, como eu não sabia programar em HTML e nem que era possível criar sites no GitHub, isso foi ficando para depois.

Também, sempre que procurava tutoriais na internet, não encontrava conteúdos mais específicos ou “exóticos”, então decidi criar este site para praticar e compartilhar o que aprendo.

Agora arrumei um tempo para estudar HTML e, sinceramente, é mais simples do que parece. É como fazer um café.

Este site não utiliza CSS por escolha de estudo e foco em HTML semântico e estrutura de conteúdo.

Isso mesmo. Este site não é bonito, nem pretende ser. Do que vale um site cheio de enfeites se ele não é acessível?

Então, como fiz esse site?

Basta você seguir alguns passos bem simples que muitos esquecem.

  1. Use títulos de níveis diferentes para cada conteúdo.
  2. Em imagens, não esqueça do atributo alt. É ele que define a descrição da imagem para o leitor de tela.
  3. Não deixe botões ou links sem rótulos.

É claro que existem outros passos a serem seguidos, mas essa já é uma boa base.

Se você observar, aqui em meu site já coloquei isso em prática, pelo menos em partes.

Bem, acho que é isso! Agora você sabe um pouco sobre mim.

Objetivos do site

Este site tem como objetivo compartilhar conhecimentos através de pequenos artigos e tutoriais em áreas que estudo e tenho domínio.
